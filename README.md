# 🌐 Cisco Packet Tracer Network Design Project

## 📌 Project Overview
This project is a fully designed and configured network simulation built using Cisco Packet Tracer. It demonstrates real-world enterprise networking concepts including VLAN segmentation, inter-VLAN routing, DHCP, NAT, and connectivity testing.

The goal is to simulate a structured, scalable, and secure organizational network environment.

---

## 🚀 Key Features
- VLAN configuration for multiple departments  
- Inter-VLAN routing (Router-on-a-Stick)  
- DHCP server for automatic IP assignment  
- NAT configuration for internet access  
- Switch and router configuration  
- End-to-end connectivity testing (Ping & Web access)

---

## 🛠️ Technologies Used
- Cisco Packet Tracer  
- TCP/IP Networking  
- Routing & Switching Concepts  

---

## 🧪 Testing Performed
- PC to PC communication  
- Inter-VLAN communication  
- Internet simulation (where applicable)  
- DHCP IP allocation verification
